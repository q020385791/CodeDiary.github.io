Draft Tool

文章 HTML 草稿產生器

直接用表單建立符合目前文章系統的 HTML 草稿。缺漏的必要欄位會自動補齊， 並可選擇先存草稿，或直接發佈到分類資料夾並更新 manifest。

使用提醒

  • 建議用本機 HTTP 預覽開啟，不要直接用 file://。
  • 第一次存檔時，請選擇 repo 內的 articles 根資料夾。
  • 內文可直接輸入 HTML；若沒有 HTML 標籤，工具會自動轉成段落。
